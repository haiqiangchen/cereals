# 上位机桌面软件（谷物测试）
## 开发环境：visual studio
## 开发语言：c#
## 开发框架：.net windowform

### 主要的功能：

·（1）、串口助手实时显示数据。

·（2）、折线图显示有效数据。

·（3）、计算当前的流量值。
